# 🚚 LogisticHub Backend

This is the backend for LogisticHub — an AI-powered Supply Chain & Logistics Management System.

---

## ⚙️ Tech Stack

- FastAPI (Backend Framework)
- PostgreSQL (Database)
- SQLAlchemy (ORM)
- JWT Authentication

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- 📦 Product Management (CRUD)
- 📊 Inventory Tracking (with logs)
- 🛒 Order Management (auto stock update)
- 🔒 Protected APIs
